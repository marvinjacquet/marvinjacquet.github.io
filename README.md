# Personal academic website

Marvin Jacquet, pharmacy resident and biostatistics student at CHU Grenoble Alpes / TIMC lab.
Live at: [marvinjacquet.github.io](https://marvinjacquet.github.io)

Based on the [Bay](https://github.com/eliottvincent/bay) theme by Eliott Vincent (ISC License)

## Added features
- Bilingual support (EN/FR) with global language toggle
- Academic pages : CV, Research (filterable by type), Teaching, Blog
- Shared component styles via `_sass/_components.scss`

## Stack
- [Jekyll](https://jekyllrb.com/) static site generator
- Hosted on [GitHub Pages](https://pages.github.com/) via GitHub Actions
- Font: [Lora](https://fonts.google.com/specimen/Lora) via Google Fonts
